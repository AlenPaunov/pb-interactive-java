# Statements
The actions that a program takes, are expressed as statements

Common statements (actions / commands) include:


* **Declaring** variables
```java
int counter;
```
* **Assigning** values
```java
counter = 1;
```
* Declaring and **initializing**
```java
int counter = 1;
```
**Printing** a value
```java
System.out.println(counter);
```
**Modifying** a value
```java
counter++;
sum = a + b;
```
